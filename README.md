# DonJuan VPN

A lightweight, self-contained web UI manager for [sing-box](https://sing-box.sagernet.org/). Dynamically generates sing-box configurations, manages proxy lifecycle, handles TUN routing, and provides rule management from a single binary.

## Supported Protocols

- **vless** (transports: none, tcp, grpc, httpupgrade)
- **hysteria2 (hy2)**
- **trojan** (transports: none, tcp, grpc, httpupgrade)

## Build

### Prerequisites

- Go 1.21+

### Required Source Files

```
singbox_ui/
├── main.go        # HTTP server, handlers, startup
├── parser.go      # Config generation, protocol parsers
├── process.go     # sing-box process management
├── models.go      # Data structures
├── index.html     # Frontend (embedded in binary)
├── go.mod         # Go module definition
└── go.sum         # Dependency checksums
```

All 7 files are required to build. `go.sum` is generated by `go mod tidy` but must be present.

### Build Command

```bash
cd singbox_ui
go build -ldflags="-s -w" -o donjuan
```

This produces a single static binary `donjuan` with the frontend embedded. **Go is only needed at build time.** The compiled binary has no runtime dependencies.

## Run

### Prerequisites

- Linux (Debian, Ubuntu, RedHat, OpenWrt)
- [sing-box](https://sing-box.sagernet.org/) installed on the system
- Root privileges (required for TUN interface and process management)

### Required Runtime Files

```
./donjuan          # The compiled binary (only file needed)
```

Everything else is created at runtime:

| File | Created When | Purpose |
|---|---|---|
| `data.json` | First run | UI state, server configs, routing rules, language preference |
| `config.json` | On start proxy | Generated sing-box configuration |
| `data/` | On geodata download | Cached `.srs` rule sets |
| `data/*.log` | When save logs is ON | Daily log files |

### Start

```bash
sudo ./donjuan
```

Open **http://127.0.0.1:8080** in your browser.

### Port

Default port is `8080`. Change it in **Settings > Panel** then restart.

## DNS Architecture

DNS resolution uses two servers:

- **remote-dns** (`8.8.8.8`, UDP) — resolves via the proxy tunnel by default
- **direct-dns** (`223.5.5.5`, UDP) — resolves proxy server hostnames and direct-bound domains directly

When TUN is active, all DNS traffic is hijacked by sing-box and routed through the proxy, preventing ISP DNS interference. Proxy server hostnames resolve via direct-dns to avoid circular dependencies.

## Multi-Language

Supported languages: English, Español, 中文, Русский.  
Select in **Settings > Panel**. The choice is saved in `data.json` and persists across restarts.

## FAQ

**Do I need Go installed to run DonJuan VPN?**  
No. The `donjuan` binary is statically compiled and self-contained. Go is only needed if you want to build from source.

**How do I update sing-box?**  
Settings > Panel > enter a version number > click Update Version.

**Why doesn't TOR work on a fresh install?**  
Make sure these defaults are ON in Core Settings: **TUN**, **Fake IP**, **Sniffing**, **Local Network**. These are enabled by default. If they were turned off at any point (saved in `data.json`), delete `data.json` and restart to restore defaults.
